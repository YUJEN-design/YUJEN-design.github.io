01 — Overview

Making healthy routines easier to maintain.

The project explores how a mobile application can reduce the friction involved in tracking wellbeing habits. The interface focuses on clear information hierarchy, calm visual language and simple daily interactions.

My role included defining the visual direction, building the interface system and designing the primary mobile screens.

02 — Challenge

Too much information can make wellness tools feel overwhelming.

Many wellness applications present users with large amounts of data, goals and notifications at the same time. This can make healthy routines feel like another demanding task.

How might we create a wellness experience that feels supportive, focused and easy to return to every day?

03 — Design approach

A simple visual system built around clarity and encouragement.

The interface uses clear typography, generous spacing and a limited set of reusable components. Each screen focuses on one primary action to reduce decision fatigue.

  • Clear daily overview
  • Simple progress tracking
  • Accessible interface hierarchy
  • Reusable component system

04 — Outcome

A calmer and more focused daily wellness experience.

The final design provides users with a simple overview of their daily activities while keeping progress visible and encouraging. The component-based system also makes future features easier to introduce consistently.
